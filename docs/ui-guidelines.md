# UI Guidelines Index

This document serves as the entry point and index for the detailed user interface guidelines, components, and design tokens of CampusLink.

---

## 1. Overview

CampusLink uses a premium dark-mode-ready, high-contrast, clean modern aesthetic. To maintain visual consistency across all modules (web and mobile WebView), refer to the detailed guidelines linked below:

| UI System Guide | Document Link | Description |

| **Design System** | [ui/design-system.md](ui/design-system.md) | Visual design tokens, color swatches (HEX & HSL), linear gradients, border-radiuses, shadows, transitions, and glow variables. |

---

## 2. Core Design Goals

* **Rich Aesthetics**: High-contrast dark backgrounds, glowing border elements, and vibrant gradient buttons to create a modern web feel.
* **Component Consistency**: Ensure every input, card, button, and loader uses the styles defined in the global stylesheet.
* **Responsive Layouts**: Designed mobile-first using grid and flex layouts that adapt to all screen sizes.
